# Directed Graph - Adjacency List + DFS

Bu bölümde bir yönlendirilmiş graf (Directed Graph) **array tabanlı adjacency list** ile gösterilmiş ve **DFS (Depth-First Search)** traversal uygulanmıştır.

## Özellikler
- Yönlendirilmiş graf
- Adjacency list kullanımı
- DFS traversal

## Örnek
Vertices: 0, 1, 2, 3  
Edges: 0->1, 0->2, 1->2, 2->0, 2->3, 3->3  

DFS başlatma: 2  
Çıktı: 2 0 1 3
