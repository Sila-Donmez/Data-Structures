# Max / Min Heap

Bu bölümde **Max Heap** ve **Min Heap** yapıları array tabanlı olarak gösterilmiştir.  

## Özellikler
- Array tabanlı heap
- Max Heap: root en büyük eleman
- Min Heap: root en küçük eleman
- Push (insert) ve pop (remove) işlemleri

## Örnek
Max Heap: [90, 15, 10, 7, 12, 2]  
- Root: 90

Min Heap: [2, 7, 10, 15, 12, 90]  
- Root: 2
